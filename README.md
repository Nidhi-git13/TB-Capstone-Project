# 🫁 TB Capstone Project

## Clinical Risk Stratification in Tuberculosis Patients using R

**Author:** Nidhi Pandey

---

## Project Overview

This project was completed as part of the R Programming Capstone Project.

The aim was to analyse clinical data from tuberculosis (TB) patients using R and answer a series of analytical questions through data cleaning, transformation, statistical analysis, and visualization.

---

## Objectives

- Import and clean clinical data
- Rename and reorganize variables
- Generate descriptive statistics
- Explore patient demographics
- Visualize clinical characteristics
- Compare medical officer and algorithm-derived risk classifications
- Identify meaningful clinical insights
